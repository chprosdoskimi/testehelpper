# Teste Helpper
## _Cadastro Usuário_

## Sobre o projeto

Este teste foi proposto pela Helpper para vaga de estágio, que resumia desenvolver um sistema de cadastro de usuários, listando e deletando os mesmos.

## Features

  - [x] Cadastrar Usuário
  - [x] Listar Usuário
  - [x] Deletar Usuário
  - [x] Atualizar Usuário

## Pré-requisitos

  - Qualquer Navegador Web
    - Chrome
    - Mozilla Firefox
    - Safari
    - Edge
